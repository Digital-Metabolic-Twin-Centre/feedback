## Overview

The IMDhub is the data management platform for the Recon4IMD project, which aims to accelerate diagnosis and personalize the management of inherited metabolic diseases by developing clinical decision support tools that integrate genomic, proteomic, and metabolomic data through computational models.

## Support & Contact

### Getting Help

For technical support, questions, or contributions:

- **GitLab Issues** - Report bugs and feature requests via [GitLab Issues](https://gitlab.com/recon4imd/imdhub/-/issues)
- **Documentation** - Comprehensive guides available via the [Wiki Page](https://imdhub.org/wiki)


### Reporting Issues

When reporting issues, please include:
1. Clear description of the problem
2. Steps to reproduce
3. Expected vs actual behavior
4. Environment information (OS, browser, versions)
5. Error logs or screenshots
6. Any attempted solutions

### Security Vulnerabilities

**DO NOT** report security vulnerabilities through public pages.

For security concerns:
- Contact the security team directly
- Provide detailed information about the vulnerability
- Allow reasonable time for remediation before public disclosure
- Follow responsible disclosure practices

### Community Guidelines

- Be respectful and inclusive
- Provide constructive feedback

- Keep discussions on-topic

### Feature Requests

We welcome feature requests! Please:
