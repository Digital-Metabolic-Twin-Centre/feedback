# Architecture Diagrams

The project architecture workspace is defined in [workspace.dsl](./workspace.dsl).

Included Structurizr views:

- `system-context`
- `containers`
- `docker-deployment`

You can open the DSL file with Structurizr Lite or the Structurizr DSL tooling to render the diagrams.

Run Structurizr Lite from this folder with:

```bash
docker compose up
```

Then open `http://localhost:8080`.
